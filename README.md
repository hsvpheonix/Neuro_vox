Description:-
**NeuroVox** is an AI-powered assistive system that converts brainwave (EEG) signals into text and speech, enabling paralyzed or speech-impaired individuals to communicate. It uses machine learning and signal processing to interpret neural patterns and generate real-time voice output. NeuroVox aims to restore independent communication through brain-to-voice technology.
